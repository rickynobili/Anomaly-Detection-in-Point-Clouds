# Anisotropic Denoising of 3D Point Clouds by Aggregation of Multiple 
# Surface-Adaptive Estimates
========================================================================
Zhongwei Xu, Alessandro Foi
v1.0.1, 2020-01-02
updates: support for CPU multicore parallel computing


This is a demo implementation of the algorithm proposed in 
Z. Xu and A. Foi, "Anisotropic Denoising of 3D Point Clouds by 
Aggregation of Multiple Surface-Adaptive Estimates",
IEEE Transactions on Visualization and Computer Graphics, 2019.
http://doi.org/10.1109/TVCG.2019.2959761


Instructions:
-----------------------------
Run 'demo_pcd_denoising.m' (read comments in the codes for more details). 


System requirements:
-----------------------------
The software has been tested under Ubuntu 18.04.2 LTS, Windows 10 and 
macOS Mojave running MATLAB R2019b.
